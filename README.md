# React + TypeScript + Vite

# 文件命名规范

## 目录命名

- 全部使用 kebab-case
- 示例: `api-routes/[, ](file://packages/client/src/lib/api-client.ts#12#59)chat-components/`

## 文件命名

- **React 组件**: kebab-case (推荐) 或 PascalCase
- **工具函数**: kebab-case
- **配置文件**: kebab-case
- **类型定义**: kebab-case
- **Hook 文件**: camelCase 或 kebab-case

## 导出命名

- **组件**: PascalCase
- **函数**: camelCase
- **常量**: UPPER_SNAKE_CASE
- **类型**: PascalCase
